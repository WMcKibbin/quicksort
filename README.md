# Quicksort
Python Quicksort implementation

#Usage
```
python qsort.py -n <list> <of> <integers> ...

or

python qsort.py -a <list> <of> <strings>
```
To use a file as input, just use inline back ticks

```
python qsort.py -a `cat strings.txt`
```
